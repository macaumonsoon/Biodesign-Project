核心概念（30 秒版）

Core concept (30-second pitch)

问题：物种灭绝带走的不仅是 DNA，还有其在昼夜与季节中的时间生态位；公众记忆常被压成图像，而非「在世界中的时间」。

Problem: extinction removes not only DNA but temporal niche—daily and seasonal rhythms in the world; memory becomes flat imagery instead of “time-in-the-world.”

方案：WebXR 纪念空间，重建猛犸象与袋狼等案例的节律与感官线索；界面区分 Cited / Interpolated / Speculative，并设伦理分叉与反思环节。

Approach: a WebXR memorial reconstructing rhythms & sensory cues (e.g. mammoth, thylacine); layers tagged Cited / Interpolated / Speculative, plus ethics forks and reflection.
