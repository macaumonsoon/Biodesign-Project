## 1. Method: how to research any taxon on the list

1. **Conservation narrative & threats (modern frame)**  
   - [IUCN Red List](https://www.iucnredlist.org/) — search by scientific name; use assessment text and citations as **cited** narrative where available. Assessment URLs can change; prefer the search URL in the CSV if a deep link breaks.

2. **Georeferenced material (map / “last known” layers)**  
   - [GBIF](https://www.gbif.org/) — species search; inspect **dataset** and **basisOfRecord** (fossil vs human observation vs preserved specimen). Treat coordinates as **cited** only when the publishing institution and catalogue number are shown.  
   - [Paleobiology Database](https://paleobiodb.org/) — especially Pleistocene megafauna; data are **CC BY 4.0** (cite the database release).

3. **Historical description & plates (smell, habitat, exploitation)**  
   - [Biodiversity Heritage Library (BHL)](https://www.biodiversitylibrary.org/) — full-text search; many volumes are public domain; always check the **volume rights** statement on BHL.

4. **Genomics / aDNA / morphology papers**  
   - [PubMed](https://pubmed.ncbi.nlm.nih.gov/) / [Europe PMC](https://europepmc.org/) — use for **cited** layers (clock genes, isotopes, CT scans, etc.).

5. **Bird sound (where recordings exist)**  
   - [xeno-canto](https://xeno-canto.org/) — per-recording **Creative Commons** licence; read the track page.  
   - [Macaulay Library](https://www.macaulaylibrary.org/) — Cornell; **per-asset** terms for reuse.

6. **Moving image (rare; flagship cases)**  
   - **Thylacine:** [National Film and Sound Archive of Australia (NFSA)](https://www.nfsa.gov.au/) holds the best-known captive footage; **licensing is restricted** — contact NFSA for exhibition or derivative use.

7. **Iconic museum specimens (narrative + AR anchors)**  
   - Treat museum web pages as **interpretation**; for **hard citation** use the collection database (catalogue number) when the institution publishes it (e.g. Smithsonian USNM numbers, Oxford Dodo specimen narrative).

## 3. Flagship archival leads (verified deep links)

Use these as **high-confidence starting points**; still verify licensing before embedding in WebXR or commercial decks.

### *Thylacinus cynocephalus* (Thylacine)

- **Full curated set:** 17 rows in `archival_media_research.csv` (NFSA, TMAG, NMA, ALA, GBIF, genomics papers, skeletal atlas, Trove, TIGRR Lab, etc.).  
- **IUCN Red List:** [species assessment](https://www.iucnredlist.org/species/21866/21949291) (IDs may be updated by IUCN — if broken, use Red List search).  
- **Moving image:** [NFSA — last known footage item](https://www.nfsa.gov.au/collection/item/tasmanian-tiger-last-footage-thylacine); [Hobart 1932 item](https://www.nfsa.gov.au/collection/item/hobart-1932-thylacine).  
- **Sound:** historic wax-cylinder–era material exists in literature; confirm rights via **TMAG / national sound archives** before claiming “cited” in UI.

### *Ectopistes migratorius* (Passenger pigeon)

- **Full curated set:** 15 rows (Martha, Birds of the World, Project Passenger Pigeon + P3 archives, BHL, two Internet Archive monographs, Audubon, xeno-canto, SI Open Access, etc.).  
- **Specimen narrative:** [NMNH — Martha overview](https://naturalhistory.si.edu/research/vertebrate-zoology/birds/collections-overview/martha-last-passenger-pigeon).  
- **SI object metadata:** [Smithsonian object record](https://www.si.edu/object/martha-passenger-pigeon%3Asiris_sic_11640).  
- **Literature / public memory:** BHL + Internet Archive host historical hunting and market literature (search from CSV).

### *Mammuthus primigenius* (Woolly mammoth)

- **Full curated set:** 14 rows (GBIF, PBDB, *Science* isotope-tusk paper, Smithsonian Magazine, Cell-lineage 3D genome PMC, bioRxiv elephantid genomes, NHM Data Portal, MorphoSource, iDigBio, GBIF extinct-species blog, etc.).

- **Paleo localities:** Paleobiology Database taxon check (URL in CSV).

## 4. Species without “famous” footage — what to do

For many Australian bandicoots, reptiles, and invertebrates on the list, **open audiovisual archives are sparse**. Recommended pipeline:

1. Use **GBIF** and **museum collection portals** (often via GBIF dataset links) for **preserved specimen** images.  
2. Use **BHL + PubMed** for **written** behaviour and habitat.  
3. Add **Sketchfab** or institutional **3D meshes** only after verifying **licence** (your CSV already notes Sketchfab search strings).  
4. Mark reconstructions from illustration or AI as **speculative** in the epistemic UI.
